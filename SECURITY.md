# Security Policy

## Supported versions

Only the latest `main` branch is actively maintained.

## Reporting a vulnerability

Please report vulnerabilities privately.

- Open a GitHub Security Advisory for this repository, or
- Contact the maintainer directly before public disclosure.

When reporting, include:

- Impact summary
- Reproduction steps
- Affected file(s) and function(s)
- Suggested remediation (if available)

Please do not open public issues for secret leaks, API keys, or exploitable flaws.
